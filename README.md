# DS1011-FinalProject
NLP Final Project

## Requirements
- python 2.7
- tensorflow (1.2.1)
- keras (2.1.2)
- numpy
- scipy
- gensim
- tqdm
- pattern
- nltk
- pandas
- scikit-learn
- h5py

To get Glove pre-trained embeddings:

$ wget http://nlp.stanford.edu/data/glove.840B.300d.zip

Then unzip the file and save at the 'embeddings' folder.
